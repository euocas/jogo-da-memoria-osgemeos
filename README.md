# Jogo da Memória – Os Gêmeos

Projeto desenvolvido como parte do **Bootcamp de Front-End da DIO (Digital Innovation One)**.

Este projeto consiste em um jogo da memória inspirado na arte dos grafiteiros **Os Gêmeos**, desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web utilizando HTML, CSS e JavaScript.

## Demonstração

**Acesse o projeto:**
https://euocas.github.io/jogo-da-memoria-osgemeos/

---

## Sobre o projeto

O jogo possui cartas embaralhadas aleatoriamente e desafia o jogador a encontrar todos os pares correspondentes. Durante o desenvolvimento foram aplicados conceitos de manipulação do DOM, lógica de programação, controle de estado da aplicação e tratamento de eventos.

---

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6)

---

## Funcionalidades

* Embaralhamento automático das cartas.
* Comparação entre pares de cartas.
* Controle de cartas viradas.
* Bloqueio de ações durante a comparação.
* Verificação de vitória.
* Interface responsiva.

## Como executar localmente

1. Clone este repositório:

```bash
git clone https://github.com/euocas/jogo-da-memoria-osgemeos.git
```

2. Acesse a pasta do projeto:

```bash
cd jogo-da-memoria-osgemeos
```

3. Abra o arquivo `index.html` em seu navegador ou utilize a extensão **Live Server** do Visual Studio Code.
